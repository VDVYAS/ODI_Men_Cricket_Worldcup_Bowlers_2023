# ODI Men's Cricket World Cup 2023 Bowlers Statistics

This repository contains data and scripts to visualize key statistics of bowlers in the ODI Men's Cricket World Cup 2023.

## Dataset

The dataset used for this analysis contains information about the bowlers participating in the Cricket World Cup 2023, including details such as wickets taken, runs conceded, boundaries given, and team-wise performance.

### Data Source
The dataset used in this analysis was collected from [source link or description].

## Requirements

To run the scripts and visualize the data, the following libraries need to be installed:

- Python (version >= 3.6)
- pandas
- matplotlib
- numpy

## Visualization

### Bar Charts

This repository includes Python scripts to generate bar charts for the following statistics:

1. Highest Wicket Taker Bowler
2. Bowler with the Highest Runs Conceded
3. Bowler who has given the Most 4s and 6s
4. Teams giving the Highest and Lowest Runs and Wickets

The visualizations are created using matplotlib.
